# LoongArch64-MikuOS

LoongArch64 OS written in Rust

## Usage

### Setup Rust Develop Environment

```bash
make setup
```

### Build

```bash
make build
```

### Run

```bash
make run
# or simply make
```

### Debug

```bash
make debug
```

Then at the same dir:

```bash
make connect
```
